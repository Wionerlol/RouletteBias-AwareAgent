A. Recent Draws *

输入你能确认的最近开奖数字，按从早到晚顺序。00 用 "00" 表示。

B. Table Stats（可选）

桌面显示屏给出的统计，样本量通常未知，仅供 agent 参考。

如果不知道样本量就留空——agent 会更保守地处理这些数据。

C. Custom Wheel Order（可选）

如果你知道这张桌的具体轮盘顺序，输入后 agent 的扇区偏向分析会更精确。

按顺时针方向列出所有数字，逗号分隔。00 用 "00" 表示。留空则使用标准美式/欧式顺序。
